### Aio

Test web application with acyncio, aiohttp and aiopg. React frontend. 

### Prerequisites

VirtualBox with GuestAdditions
Vagrant

## Deployment

### Clone repository

```
git clone https://github.com/TeaTracer/aio-test.git
cd aio-test
```

###  Vagrant

```
vagrant up
vagrant ssh
/vagrant/server/deploy/script.sh
```
